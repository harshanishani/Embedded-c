# Embedded-C

🚀 My Embedded C learning journey and practice repository.

This repository contains Embedded C programs, bit manipulation techniques, pointers, memory concepts, communication protocols, and firmware-related exercises that I learn and practice daily.

The goal of this repository is to strengthen low-level programming skills and build a strong foundation in Embedded Systems and Firmware Development.

# 📚 Topics Covered

- C Programming Fundamentals
- Bitwise Operators
- Bit Manipulation
- Pointers
- Structures & Unions
- Memory Management
- Storage Classes
- Arrays & Strings
- Function Pointers
- Embedded C Concepts
- UART / SPI / I2C Basics
- RTOS Basics
- Microcontroller Programming
- Register-Level Programming
